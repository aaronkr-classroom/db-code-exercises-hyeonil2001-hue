# chp3- 실습과제.md

## 데이터 요구 분석 명세서 (각3개이상)

### 1. 여행상품 관리

- 새로운 여행상품을 계속 개발하고 있다
- 여행상품을 국가별로 분류한다
- 대륙별로 3개 이상의 코스 여행상품을 만들 계획이다

### 2. 예약 관리

- 고객이 여행상품을 예약한 날짜와 예약금 납부 여부를 관리한다
- 예약은 먼저 납부가 되어야 예약 확정이 가능하다
- 예약일 1달 전에 환불할 시에만 전액 환불이 가능하다

### 3. 고객 관리

- 고객 등급은 일반, 우수, 최우수로 구분한다
- 고객의 이름, 비밀번호, 주소를 관리한다
- 5년 이상 장기 미활동 고객은 휴면 고객으로 전환한다

### 4. 직원 관리

- 현재 약 30명의 직원이 근무하고 있다
- 직원들을 직급별로 분류한다
- 직원은 고객과 별도의 고유번호를 가진다

## 트랜잭션 요구 분석 명세서 (crud 포함가능)

### 1. 여행상품 관리

- 직원은 자신이 담당하는 여행상품을 검색할 수 있다
- 직원은 새로운 여행상품을 등록할 수 있다
- 직원은 기존의 여행상품을 삭제할 수 있다

### 2. 예약 관리

- 고객이 여행상품의 예약 일자 정보를 삽입한다
- 직원은 고객의 예약 기록을 검색할 수 있다
- 여행 상품이 취소된 경우 예약한 고객에게 전액 환불한다

### 3. 고객 관리

- 탈퇴한 고객 정보를 삭제한다
- 고객은 자신의 개인정보를 변경할 수 있다
- 고객은 유효한 예약이 없는 상태에서 스스로 탈퇴할 수 있다

### 4. 직원 관리

- 직원은 자신의 휴대폰 정보를 변경할 수 있다
- 신규 직원의 정보를 입력한다
- 퇴사한 직원의 정보를 삭제한다
